Introduction:

 

This user guide is aimed to provide a synopsis of the Petrol Station Visualization Software, along with providing all the essential information, regarding the use of the Application.

 

Petrol Station Visualization Software, is a program built to simulate the events of a normal, yet exceptionally busy (cars arrive every 1-2 seconds) day in a Petrol Station, during the holiday season. The application is able to create a minimalistic, yet informative graphical interface for the user, where vehicles arrive, wait in the queue, get assigned to the pumps, and finally, they get served by the petrol stations assistant. To be honest though, the getting serviced part is not that simple, because drivers, do tend to be impatient and leave the queue during the holidays, especially if they have kids.

 

 

 

Is this software for me?

 

Ever wondered if working in a petrol station would be a good idea for you? Or maybe wondered if owning one would be a solid investment of money? Or even if you are just the type of person, who happens to enjoy watching simplistic images pop up on the screen and follow a standardized procedure, if thats the case, think no more! This is the software for you.

Simulate, with the click of a button, the whole thrill of a petrol station everyday life. Witness the patience of the drivers waiting to be serviced run out, and the struggle of the petrol stations assistant to manage to serve as many people as possible.

 

 

Prerequisites:

·         .Net Framework

·         Release folder of the project

 

 

Step-By-Step Guide:

1.      Open the release folder of the project.

2.      Double click on the ProgrammingAssignement2017.exe

3.      Press on the Play button to start the simulation.

 

 

4.      Initially you will notice the nine Pumps and the empty queue.

5.      Vehicles will start spawning and populating the available spots.

            Van                     Car                       HGV

6.      Hover over the cars getting served in the pumps to see what kind of fuel they are using.

7.      Press the Show More button to see all the counters going up while the petrol stations assistant is serving the vehicles.

8.      Press the Show Less button to hide all the counters, to enjoy a cleaner User Interface.

9.      Press the Pause button to stop the simulation momentarily. Start it again by pressing on Play, whenever you are ready to start again!

 

10.  Press the Stop button whenever you are ready to stop the simulation completely. On the right side of the form, a list of all the transactions completed will present itself.

11.  Press on the X on the top right side of the form to close it at any moment. A message asking you if you want to save the details of you simulation will pop up.

12.  Press yes to save a detailed list of all the cars that spawned and reached the queue of the petrol station and then exit.

13.  Press no to exit without saving,

14.  Press cancel to return to the simulation.

15.  To see the details of your simulation that you saved, go to the release folder and open the PetrolStationData folder. Inside, you will find all the saved sessions, each named with the following format Date-Time of the simulation.

16.  Enjoy!

 

 

Troubleshooting

 

·         Make sure that you have a recent version of the .Net framework.

·         Make sure that you are using the .exe file, from inside the release folder.
